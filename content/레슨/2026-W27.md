# 레슨 — 2026-W27

## 배운 점

- **PowerShell 보안 정책** — npm 스크립트 실행 시 `Set-ExecutionPolicy RemoteSigned` 필요
- **git 서브모듈 함정** — 외부 git 레포를 클론하면 서브모듈로 인식됨. `.git` 폴더 제거 후 강제 추가 필요
- **Cloudflare Pages Node 버전** — `NODE_VERSION` 환경변수로 버전 지정 가능
- **npm engine 체크** — `.npmrc`에 `engine-strict=false` 추가하면 버전 불일치 경고로 처리
- **인증 코드는 채팅창 말고 터미널에** — Claude Code 로그인 코드를 채팅창에 붙여넣으면 보안 위험
- **배포 먼저, 수정은 나중** — 완벽하지 않아도 먼저 배포하고 수정해가는 방식이 효율적
- **PDF 자료 → 홈페이지** — 기존 자료(PDF, PPT)가 있으면 Claude가 읽고 바로 홈페이지로 변환 가능

## 막혔던 순간
1. `nvm` 명령어 인식 안 됨 → PowerShell 재시작으로 해결
2. Quartz 플러그인 서브모듈 문제 → `.git` 폴더 제거 + 강제 추가
3. Cloudflare 빌드 실패 반복 → `.npmrc` + `NODE_VERSION` 조합으로 해결
4. 네트워크 공유 폴더 파일 → 로컬 복사 후 Claude에게 전달
