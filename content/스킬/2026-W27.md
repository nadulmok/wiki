## 새로 배운 것
- **네트워크 공유 폴더 접근** — `\\서버명\폴더` 형식으로 PowerShell에서 접근 가능
- **PDF → 홈페이지 변환 흐름** — PDF 내용 분석 → 섹션 구조화 → HTML 제작
- **GitHub Pages 배포 흐름** — `git init` → `commit` → `push` → Settings > Pages 설정

## 새로 배운 명령어
```powershell
copy "\\서버\폴더\파일" "$env:USERPROFILE\로컬경로"   # 네트워크 파일 복사
git init                   # git 저장소 초기화
git remote add origin URL  # 원격 저장소 연결
git push -u origin master  # 최초 push
```
